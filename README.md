
# Bloczki-Gra-PWaG

A voxel engine/game project for PwAG subject (Programowanie w API Graficznych)

## Controls

WSAD - standard movement

Space/Shift - Up/Down

Left/Right click - Break/Place blocks

Mouse wheel - change block type

Arrow keys left/right - change in-game time of the day

#### Debug Controls
F1-F12 - change render mode

F1 - Default

F2 - Albedo

F3 - Normals

F4 - Occlusion

F5 - Roughness

F6 - Metallic

F7 - Skylight level

F8 - Blocklight level

F9 - Emissive

F10 - Depth

F11 - Bloom

F12 - Wireframe
